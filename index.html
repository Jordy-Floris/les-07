Reflectie

Bij de lessen HTML & CSS heb ik mij meer kunnen verdiepen in deze tools. Ik wist van eerdere ervaring al wat bepaalde tags betekenden zoals H1, P, img, enz maar in de lessen leerde ik meer over de exacte regels zoals waarom je Section tags moet gebruiken en hoe belangrijk het is voor jezelf en voor anderen dat je georganiseerd te werk gaat. Ook leerde ik hoe ik ik mijn HTML websites online kon zetten met behulp van Github.

Actiepunten

  • 1. Maak een nieuw bestand in Visual Studio Code
  • 2. Sla dit bestand op als index.html
  • 3. Defineer het bestand als DOCTYPE HTML
  • 4. Voeg de content van je website toe (H1, P, IMG, ect.)
  • 5. Maak je CSS bestand aan als style.css
  • 6. Link je CSS met je HTML met de link tag
  • 7. Style je HTML in je CSS bestand met selectoren, id's en classes
  • 8. Bekijk je website in de browser en gebruik de inspecttool om hem te verbeteren
